# Screenshots

Drop PNG files in this directory and reference them from the main README.

Suggested file names (any will do, just keep them consistent):

- `menu-bar.png` — the shield icon in the macOS menu bar
- `settings-panel.png` — the dark glass settings popover
- `countdown.png` — the 10-second countdown HUD
- `walk-away.png` — the walk-away blur overlay
- `peek.png` — the over-the-shoulder peek blur overlay
- `unlock.png` — the Touch ID prompt on top of the blur

How to capture screenshots on macOS:

- Cmd+Shift+4 then Space, then click a window to capture just that window with shadow.
- Cmd+Shift+5 for the screenshot toolbar with timed and recording options.

For full-screen blur shots, use a second monitor or capture the whole screen with Cmd+Shift+3.
